Property — class, package axl.utils.liveArrange
Class prepared to parse and modify single node of XML description of object defined as editor.subject of this instance. Contains two textfields: name and value. Allows bi-directional update: updateFromObjectToTextfield - from object's actual value to this instance text input field updateFromTextfieldToObject - from this instance text input field to actual object Elements are parsed through JSON.parse if needed. Class has no internal event listeners.
