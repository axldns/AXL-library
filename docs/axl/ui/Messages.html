msg()method
public static function msg(v:String = null, onTapOutside:Function = null, onTapInside:Function = null, forceClickInside:Boolean = false):void

Displays TextField with html text on top of Stage. TextField is removed after user's click/touch.

Parameters

v:String (default = null) — - text/html text to display. If null - removes TextField from stage
 
onTapOutside:Function (default = null) — - callback to execute when user clicks anywhere on the stage (but message itself) while message is displayed
 
onTapInside:Function (default = null) — - callback to execute when user clicks on the message itself
 
forceClickInside:Boolean (default = false) — - determines if message will be removed after user clicks anywhere (false) or it requires user to click on the message in order to remove it (true). Default: false
