Classpublic class Scroll
InheritanceScroll flash.display.Sprite

Class provides simple scroll bar functionality. It makes use of axl.ui.controllers.BoundBox controler, which needs two Display Objects to satisfy functionality (bound & box), here named as rail and train. Scroll bar can have only one orientation at the time. If scroll bar requires buttons, these should call increase and decrease functions for guaranteed results. No manual position validation is needed in this case. Controller's initial values can be overriden, default values as follows: ctrl.bound = rail;ctrl.box = train;ctrl.horizontalBehavior = BoundBox.inscribed; ctrl.verticalBehavior = BoundBox.inscribed;ctrl.liveChanges = true;
