Packageaxl.ui
Classpublic class Carousel
InheritanceCarousel flash.display.Sprite

Class provides carousel component. Ideal for scrollable lists galleries etc. Supports children of different dimensions. Can be horizontal OR vertical. Axles can be switched seamlessly (with children inside). Set x and/or y properties of this object exactly where you want to the see center of your carousel (as carousel would be 1px wide and/or 1px high). Children are being distributed from the most center point. Children indices are under control of this class. Component is well optimized: whenever possible - updates inner container (railElementsContainer) rather than each and every child separately. Class overrides standard children manipulation methods (addChild, addChildAt, removeChild, removeChildren, removeChildAt, getChildByName, getChildAt, swapChildren, swapChildrenAt, getChildIndex, setChildIndex, numChildren). Requests of such will reffer to railElementsContainer, therefore using element's parent should be takien into special consideration. Class does not provide animation or 'next element selection' utilities. Carousel provides only one method (movementBit) for propel, which accepts just one parameter (delta). All further and higher level features should be added separately e.g. extending this class. Supports gaps and allows to get "MIDDLE" child.
