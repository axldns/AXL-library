Packageaxl.ui
Classpublic class MaskedScrollable
InheritanceMaskedScrollable flash.display.Sprite

Classextends Sprite by adding three functionalites:
  • provides ready to use and already assigned, rectangular mask - controllable by properties visibleWidth, visibleHeight, visibleX, visibleY
  • provides controller property - an instance of BoundBox class, which allows to scroll, drag and move masked contents within masked area
  • exposes ability to make bi-directional connections with any other BoundBox controllers, e.g. members of Scroll instance
Instance auto-recognizes Scroll instance added to it's display list, places it outside of masked container and auto-connects scroll bar's controller to MaskedScrollable controller. Since method above is just a short-hand kept for backward ccompatibility, all other controllers should be connected via connectController function. Initial controller initial values can be overriden, default values as follows: ctrl.horizontal = true;ctrl.vertical = true;ctrl.horizontalBehavior = BoundBox.described; ctrl.verticalBehavior = BoundBox.described;ctrl.liveChanges = true; Values for bound and box of controller should never be changed.

scrollBarproperty 
scrollBar:Scroll

Allows to make bi-directional connection between scrollBar instance and masked container. Bi-directional means that both scroll bar can update container and container can update scroll bar. Getter returns only most recent scroll bar added. Multiple scroll bars can be set/added.
