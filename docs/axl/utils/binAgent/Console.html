Packageaxl.utils.binAgent
Classpublic class Console
InheritanceConsole flash.display.Sprite
Subclasses BinAgent

Console class is where all framework’s logging is routed to, but it can be used completely stand-alone. It imitates real console/terminal where all output is printed line by line. Can be filtered, opened, closed, diabled. Console window can be opened in three ways:
  1. programatic: simple as isOpen = true
  2. with definable keyboard shortcut/sequence (default: ctrl+alt+right)
  3. with adjustable gesture (default: four horizontal drags in top left corner area)
It can be controlled also via allowKeyboardOpen and allowGestureOpen flags. For any commercial use, please do change default settings, since this tool can cause serious harm to your business logic. Entering to the input field text you’re looking for followed by a colon. E.g. entering :config would display only lines that are containing config chunk.

trrace()method 
public function trrace(... args):int

Behaves simmilar to native trace function - accepts number of arguments of any type, and prints out its stringified version. Main difference is output. For native trace it would be debugger console. For this function, default output is console window of this class. Differently, it prints out XML and XMLList string representation.
