Packageaxl.utils.binAgent
Classpublic class BinAgent
InheritanceBinAgent Console flash.display.Sprite

Extends Console class by adding two main functionalites:
  • Parses / evaluates input text from input window and prints out result of this parsing
  • Evauluates input "as you type" and gives IDE-style code-completion hints
