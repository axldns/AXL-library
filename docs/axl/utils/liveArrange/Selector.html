Packageaxl.utils.liveArrange
Classpublic class Selector
InheritanceSelector flash.display.Sprite

Class that holds and displays information of the display object set as target. Draws it's outline and live(onEnterFrame) tracks it's bounds. Selector listens to enterFrame events only when its added to stage. Removed, removes listeners automatically. Displays basic info (type,name,x,y,width,height) in an infobar on top of outline. If mouseEnabled = true, allows to doubleclick selection and fire onDoubleClick.
