Packageaxl.utils.liveArrange
Classpublic class LiveArranger
InheritanceLiveArranger Object

Class that allows onscreen edit display objects. Objects are being detected based on mouse pointer position. An outline of currently detected object is being drawn. Hotkeys:
  • F6 - turns it on/off.
  • PageUp,PageDown,Home - opens or closes editor window for selected object
  • hold CTRL or CMD
    • if editor is off: allows to drag buttons preventing from executing their functions
    • if editor is on: allows to select new object without exiting editor
  • Z - (traverse up) sets parent of selected object as selected object if available.
  • X - (traverse down) selects first child of selected object as selected object if available.
  • 1 - (traverse side) selects another child(+1) in container of selected object if available.
  • 2 - (traverse side) selects another child(-1) in container of selected object if available.
  • ESC - turns off editor. If editor is closed, turns aranger off.
Singletone contoll is managed by dispatching SyncEvent of type axl.utils.LiveAranger to stage.loaderInfo.sharedEvents passing itself instance in changesList. If there's any primary class listening to this event, it will call destroy method on the instance passed in changesList. If there's no response on SyncEvent, the dispatcher instance becomes ruling primary listener to SyncEvents, and it's only allowed one to continue with building it's contents.
