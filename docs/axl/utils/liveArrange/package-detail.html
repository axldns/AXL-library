axl.utils.liveArrangeClasses
 


Classes
 ClassDescription
 EditorWindow Class that allows "on screen edit" any DisplayObject. Allows to drag and resize this window aside from editing subject. Provides: list of all properties which can be set for subject of this editor Simple links to traverse display list up, down and sideways options to filter list by properties name Input fields have easing for setting numeric and boolean values: use keyboard arrows!
 LiveArranger Class that allows onscreen edit display objects.
 Property Class prepared to parse and modify single node of XML description of object defined as editor.subject of this instance. Contains two textfields: name and value. Allows bi-directional update: updateFromObjectToTextfield - from object's actual value to this instance text input field updateFromTextfieldToObject - from this instance text input field to actual object Elements are parsed through JSON.parse if needed. Class has no internal event listeners.
 Selector Class that holds and displays information of the display object set as target. Draws it's outline and live(onEnterFrame) tracks it's bounds.
