mergeLoadings()method 
protected function mergeLoadings():void

This function is in use when there are files to load from two different resources and with different behaviors. Update files need to be downloaded from web and stored to hdd Regular files need to be loaded according to defaultPathPrefixes Both need to display progress and listen for global complete. This effectively makes two queues with both separate and global listeners. On complete of update queue, new config date need to be stored.
